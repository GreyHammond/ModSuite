What's included

Everything. Out of the box.

One /setup command builds your entire server structure automatically.

📬

ModMail

Users DM the bot to open a private ticket. Staff reply with anonymous or named responses. Transcripts are zipped and archived on close.

DM → private channel
🔒

Jail System

Pull a user aside privately. Strips all roles, creates a dedicated channel, then restores everything on unjail with a full transcript.

role save & restore
🛡️

Mod Panel

A persistent button panel for staff who prefer UI over commands. Context-aware — shows different controls in jail, modmail, and general channels.

context-sensitive
🎨

Self-Roles

14 color roles users can self-assign by reacting to a message. Bot enforces one color at a time and removes unknown reactions automatically.

reaction-based
⚠️

Warn System

Track warnings per user with configurable auto-escalation. Reach the mute threshold and they're timed out. Reach the ban threshold and they're gone.

auto-escalation
🚨

Raid Protection

Monitors join rates and auto-locks the server when a raid is detected. Configurable thresholds. Manual lockdown and unlock commands included.

auto-lockdown
🚩

Message Reports

Right-click any message → Apps → Report. Standard reports ping Moderators. Emergency reports blast all staff with a hard-to-miss red embed.

right-click context menu
💾

Persistent Config

SQLite database with automatic schema migrations. Update the bot and restart — new columns are added without wiping your data.

zero data loss updates
📋

Mod Log

Every action — kicks, bans, mutes, warns, jails, lockdowns — is automatically posted to a dedicated staff-only log channel.

full audit trail
Slash commands

Commands at a glance

All commands have Discord autocomplete. Staff who prefer buttons can use /mod anywhere for the ephemeral panel.

/setup Run guided server setup — creates all roles, channels, and posts the self-roles message admin
/panel Post the persistent mod panel in a staff channel staff
/mod Open ephemeral context-aware panel anywhere — jail, modmail, or general staff
/warn @user reason Warn a member. Auto-mutes and auto-bans at configurable thresholds staff
/history @user View full moderation history — all warns, active and removed staff
/jail @user [notify] Strip roles and open a private channel. Optional DM notification flag staff
/unjail @user Release from jail and restore all original roles. Archives transcript staff
/mute @user [10m/2h/1d] Timeout a member. Duration supports d/h/m/s. Defaults to 30 days staff
/kick · /ban · /unban Standard moderation actions with optional reason and mod-log entry staff
/lockdown · /unlock Manually lock or restore all channels. Also triggers automatically on raids staff
/reply · /close Reply to a ModMail ticket (with anon flag) or close and archive it staff
/userinfo · /serverinfo User and server info. Staff see warn count and jail status everyone
/purge [amount] Bulk delete up to 100 messages in the current channel staff
Right-click → Report Report any message anonymously. Emergency report pings all staff everyone
Getting started

Up in five minutes.

ModSuite is self-hosted. You own your data, your bot, your server.

Clone the repo

Download or clone from GitHub. Requires Python 3.11+ and a Discord bot token.

Install dependencies

Run pip install -r requirements.txt. Only two packages: discord.py and python-dotenv.

Add your token

Copy .env.example to .env and paste your bot token. Enable Members and Message Content intents.

Start the bot

Run python bot.py. Commands sync to your server instantly on startup.

Run /setup

One command creates every role, channel, and message automatically. Re-run safely at any time.

Post /panel

Run /panel in a staff channel to post the persistent mod panel. Done.
